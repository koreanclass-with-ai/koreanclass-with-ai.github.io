// 들려줄 지문 — 이 글이 곧 모범답안이에요 const passage = "지난 주말에 친구하고 영화를 봤어요. 영화를 보는 동안 밖에 비가 많이 왔어요. 비가 오기 때문에 우리는 카페에서 커피를 마시면서 기다렸어요."; // 쓰기 단계의 키워드 힌트 (힌트 버튼을 누르면 보여요) const keywords = ["주말", "친구", "영화", "-는 동안", "비가 오다", "-기 때문에"]; // 음성 속도 (1이 보통, 0.85가 조금 느리게) let currentRate = 0.85; // ⚠️ 지문을 바꿀 때: 목표 표현이 자연스럽게 들어가게, // 길이는 3~4문장(중급 기준)을 지켜요③ 작동 로직

"block" : "none"; }); // ③ 비교 단계: 원문과 학생 글을 나란히 보여주기 if (n === 3) { originalText.textContent = passage; // 모범답안 myText.textContent = writeArea.value; // 학생이 쓴 글 } } // 키워드 힌트 보이기/숨기기 hintBtn.addEventListener("click", () => { keywordBox.hidden = !keywordBox.hidden; }); // '처음부터 다시' — 쓴 글과 들은 횟수 초기화 function resetDicto() { writeArea.value = ""; playCount = 0; goStage(1); }
