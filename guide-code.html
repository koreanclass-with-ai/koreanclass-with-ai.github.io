WHAT
코드 생성 도구가 뭔가요?

코딩을 직접 하는 게 아니라, AI한테 "이런 앱 만들어줘"라고 한국어로 설명하면 AI가 코드를 짜주는 방식이에요. 이걸 '바이브 코딩'이라고 해요.

활동 페이지의 Step 3에서 "코드 생성 도구에 붙여넣고"라고 한 게 바로 이 도구예요. JSON 데이터 + 원하는 기능 설명을 넣으면 학습자가 쓸 수 있는 웹앱이 나와요.

Antigravity를 추천해요. Google에서 만든 도구고, 무료 플랜으로 시작할 수 있어요.

SETUP
설치와 초기 설정

처음 한 번만 하면 돼요. 5분이면 끝나요.

1다운로드
antigravity.google/download에서 사용 중인 운영체제(Windows/macOS)에 맞는 버전을 다운로드하고 설치하세요.
2초기 설정
Start fresh 선택 → 테마 취향대로 → Review-driven development 모드 → Normal 키보드 → Recommended 확장 설치.
3로그인
구글 계정으로 로그인하세요. 무료 Individual 플랜으로 시작 가능해요.
4한국어팩 설치
왼쪽 메뉴 → 확장 기능(사각형 아이콘) → "korean" 검색 → Korean Language Pack 설치.
5작업 폴더 열기
바탕화면에 폴더 하나 만들고(예: "한국어수업앱"), Open Folder로 선택하세요. 앱 파일들이 여기에 저장돼요.
Review-driven development를 꼭 선택하세요. AI가 뭔가 하기 전에 "이렇게 할까요?" 하고 물어봐요. 코드를 모르는 상태에서 AI가 마음대로 파일을 바꾸면 당황스러우니까요.
USE
사용법

설치 끝나면 채팅창에 한국어로 원하는 걸 적으면 돼요.

프롬프트에 넣을 세 가지

① 앱 설명 — 어떤 앱인지 큰 그림. "한국어 학습용 빈칸 채우기 웹앱"

② 기능 명세 — 버튼 누르면 어떻게 되는지 구체적으로. "정답 선택 시 초록색, 오답 시 빨간색, 해설 표시"

③ 데이터 — AI가 만들어준 JSON을 그대로 붙여넣기.

예시: Antigravity 프롬프트
이 데이터로 한국어 학습용 빈칸 채우기 웹앱 만들어줘. 기능: - '에' 또는 '에서' 선택하면 즉시 정답/오답 표시 - 문항마다 해설 표시 - 10문항 다 풀면 점수 보여주기 - 문항 순서 매번 랜덤 - 모바일 반응형 데이터: [여기에 JSON 붙여넣기]
한 번에 완벽하게 나오지 않아요. "문제 순서를 랜덤으로 해줘", "버튼 크기 좀 키워줘" 같은 수정 요청을 여러 번 하는 게 정상이에요.
TIPS
실전 팁
  • 작게 시작하세요. 처음부터 기능 10개를 한꺼번에 넣지 말고, 문제 5개짜리 간단한 퀴즈부터 만든 다음 기능을 하나씩 추가하세요.
  • 에러가 나도 괜찮아요. 에러 메시지를 그대로 복사해서 "이런 에러가 나는데 고쳐 줘"라고 하면 알아서 고쳐줘요. 에러가 뭔 뜻인지 몰라도 돼요.
  • 미리보기를 자주 확인하세요. 수정할 때마다 브라우저에서 결과를 확인하고, 학생 입장에서 직접 눌러보세요.
  • 모바일 테스트는 필수예요. 학생들이 스마트폰으로 접속하니까, 폰에서도 잘 되는지 확인하세요.
  • 완성되면 파일을 백업하세요. 폴더에 생긴 HTML 파일이 곧 완성된 앱이에요. 다른 곳에도 복사해 두세요.
