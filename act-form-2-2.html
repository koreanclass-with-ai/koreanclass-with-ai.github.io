빈칸 채우기는 뭐고, 변환은 뭐냐고요? 빈칙 채우기가 문장의 일부를 채우는 거라면, 변환은 문장 전체를 다시 쓰는 거예요. "저는 매일 운동해요" → "저는 어제 운동했어요"처럼 학습자가 시제 판단 + 어미 활용 + 시간 부사 교체를 동시에 수행해야 하거든요.

왜 변환이 시제를 가르치기에 좋을까요? 시제는 "같은 사건이 시점에 따라 어떻게 표현되는가"가 핵심이에요. 같은 문장의 현재형과 과거형을 나란히 경험하면, 학습자가 "무엇이 바뀌고 무엇이 그대로인가"를 포착하기 쉬워요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
불규칙 활용 문항은 규칙 활용 문항 사이에 섞어서 배치해야 해요. 불규칙만 연속으로 나오면 학습자가 "이건 다 불규칙이겠지"라고 예측하고, 규칙 적용의 판단을 건너뛰거든요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 2급 학습자를 위한 시제 변환 문항 생성 # 조건 (Constraints) - 문항 수: 10개 - 각 문항: 현재 시제(-아요/어요) 문장 1개를 제시 - 학습자가 과거 시제(-았어요/었어요)로 변환 - 불규칙 활용 2개 이상 포함 (ㄷ불규칙: 듣다→들었어요, ㅂ불규칙: 춥다→추웠어요 등) - 학습자가 시간 부사("어제", "지난주에")를 직접 추가하도록 지시 # 출력 (Output) JSON 형식
역할과 출력 형식은 그대로 두고, 과제와 조건만 바꾸면 어떤 문법 항목이든 만들 수 있어요.
🔁 이렇게 바꿔서 응용하세요

시제 변환 대신 다른 변환 활동으로 만들고 싶으면, 과제와 조건만 수정하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
변환 방향을 바꾸세요. "반말→존댓말 변환", "평서문→의문문 변환", "능동→피동 변환" 등으로요.
조건
레벨에 맞게 조정하세요. 중급이면 세 시제(현재/과거/미래) 동시 변환도 가능해요.
출력
앱으로 만들 거면 JSON, 인쇄용이면 "표 형식으로 정리해줘"로 바꾸세요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 문항을 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "questions": [ { "id": 1, "present": "저는 매일 운동해요.", "past": "저는 어제 운동했어요.", "verb": "운동하다", "irregular": false }, { "id": 2, "present": "음악을 들어요.", "past": "어제 음악을 들었어요.", "verb": "듣다", "irregular": true }, { "id": 3, "...": "..." } ] }
present
→현재 시제 문장이에요. 이 문장이 학습자에게 제시돼요. (green)
past
→과거 시제로 변환된 정답 문장이에요. (blue)
verb
→핵심 동사의 기본형이에요. 해설에 사용돼요. (purple)
irregular
→불규칙 활용 여부예요. true면 불규칙 규칙이 적용돼요. (pink)
⚠️ AI가 만든 문항은 꼭 확인하세요. 어색한 표현이나 수준에 맞지 않는 어휘가 섞여 있을 수 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
🛠 Step 3. 코드 생성 도구로 앱을 만드세요

JSON 데이터가 준비됐으면, Antigravity 같은 코드 생성 도구에 붙여넣고 이렇게 말하면 돼요.

"이 데이터로 한국어 학습용 시제 변환 웹앱 만들어줘. 현재형 문장을 보고 과거형 문장 전체를 직접 타이핑하면, 정답과 비교해서 맞으면 초록색, 틀리면 빨강색으로 표시되고, 정답과 함께 동사의 불규칙 활용 여부를 설명해줘. 10문항 다 풀면 점수 보여줘."

완성되면 아래처럼 작동하는 앱이 나와요. 직접 눌러보세요!
