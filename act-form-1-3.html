AI로 활용 연습 앱 만드는 법

프롬프트 하나로 시제·어미 활용 문항을 뽑고, 자동채점 앱까지 만드는 과정을 단계별로 안내해요.

설계 의도 제작 과정 연습 문제 점검·응용
INTRO
💡 왜 이 형식으로 만들었나요?
활동 유형빈칸 채우기 (직접 입력형)
범주❶ 형태 학습
대상초급~초중급 (TOPIK 1~2급)
목표동사를 적절한 어미로 활용하여 문장 완성
언어 기능읽기 · 쓰기
규모10문항 / 10분
배포 형태웹 기반 또는 인쇄용 워크시트

선택형이 "보기에서 고르는" 활동이라면, 직접 입력형은 "직접 써내는" 활동이에요. 보기 없이 정답을 스스로 산출해야 하니까 인지적 부담이 한 단계 높아지죠. 선택형에서 충분히 연습한 뒤, 같은 문법을 이 형식으로 다시 연습하면 "아는 것"에서 "쓸 수 있는 것"으로 전환이 촉진돼요.

왜 문장 완성이 어미 활용에 적합하냐면요. 어미 활용은 기본형에서 활용형을 "만들어내는" 과정이잖아요. "만나다"를 보고 "만났어요"를 써내려면, 학습자는 시제 판단(어제 → 과거) + 활용 규칙 적용(만나 + 았 + 어요) + 철자 확인이라는 연쇄 과정을 거쳐야 해요.

문장 완성은 이 연쇄 과정을 빈칸 하나에 압축시켜 연습시키는 가장 효율적인 형식이에요. 시간 부사("어제", "내일", "지금")가 시제를 판단하는 단서 역할을 하거든요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 1~2급 학습자를 위한 문장 완성 문항 생성 # 조건 (Constraints) - 문항 수: 10개 - 각 문장에 동사 기본형을 괄호로 제시, 학습자가 적절한 어미로 활용 - 목표 어미: -아요/어요 (현재), -았어요/었어요 (과거), -(으)ㄹ 거예요 (미래) 균형 배치 - 불규칙 활용 포함: ㄷ불규칙, ㅂ불규칙 각 1개 이상 - 문맥(시간 부사 등)으로 시제를 판단할 수 있도록 구성 # 출력 (Output) JSON 형식
시간 부사("어제", "내일", "지금")를 단서로 넣으면 학습자가 시제를 판단한 뒤 어미를 활용하는 2단계 사고가 유도돼요. 시간 부사를 빼면 난이도가 올라가요.
정답 표기를 해요체로 통일할지 합쇼체로 통일할지 미리 정해야 해요. "만났어요"와 "만났습니다"가 모두 정답이 되면 자동 채점이 복잡해지거든요.
🔁 이렇게 바꿔서 응용하세요

어미 활용 대신 다른 문법 항목으로 만들고 싶으면, 과제와 조건만 수정하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
목표 어미를 바꾸세요. "연결어미 활용 문항 생성", "종결어미 활용 문항 생성" 등으로요.
조건
레벨과 특성에 맞게 조정하세요. 중급이면 TOPIK 3급, 불규칙 비율을 높이거나 시간 부사를 빼서 난이도 조절해요.
출력
앱으로 만들 거면 JSON, 인쇄용이면 "표 형식으로 정리해줘"로 바꾸세요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 문항을 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "questions": [ { "id": 1, "sentence": "저는 어제 친구를 ______. (만나다)", "answer": "만났어요", "tense": "과거", "clue": "어제", "irregular": false }, { "id": 2, "sentence": "내일 날씨가 ______. (춥다)", "answer": "추울 거예요", "tense": "미래", "clue": "내일", "irregular": true }, { "id": 3, "...": "..." } ] }
sentence
→빈칸과 기본형이 포함된 문장이에요. 학습자가 이 문장을 보고 활용형을 입력해요.
answer
→빈칸에 들어갈 정답 활용형이에요. 해요체로 통일돼 있어요.
tense
→해당 문항의 시제예요. 현재/과거/미래 균형 배치를 확인하는 데 써요.
clue
→시제를 판단할 수 있는 단서(시간 부사 등)예요.
irregular
→불규칙 활용 여부예요. true이면 ㄷ/ㅂ/ㅅ 불규칙 등이 적용돼요.
⚠️ AI가 만든 문항은 꼭 확인하세요. 불규칙 활용 문항의 정답이 틀려 있거나, 시간 부사와 시제가 맞지 않는 경우가 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
🛠 Step 3. 코드 생성 도구로 앱을 만드세요

JSON 데이터가 준비됐으면, Antigravity 같은 코드 생성 도구에 붙여넣고 이렇게 말하면 돼요.

"이 데이터로 한국어 어미 활용 문장 완성 웹앱 만들어줘. 문장과 기본형 보여주고, 학습자가 활용형 입력하면 정답/오답 표시되고, 시제 판단 근거도 보여줘."

완성되면 아래처럼 작동하는 앱이 나와요. 직접 풀어보세요!
