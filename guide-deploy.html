코드 생성 도구에서 만든 HTML 파일은 내 컴퓨터에만 있어요. 학생이 접속하려면 인터넷에 올려야 해요. 어렵지 않아요.

방법 1: Antigravity에서 바로 배포

Antigravity로 만들었다면 내장 배포 기능으로 바로 URL을 받을 수 있어요. 가장 간단한 방법이에요.

방법 2: Netlify 드래그 앤 드롭
1netlify.com 접속
무료 회원가입 후 로그인하세요.
2파일 드래그 앤 드롭
"Sites" 탭에서 HTML 파일(또는 폴더)을 브라우저 화면에 끌어다 놓으세요.
3링크 복사
몇 초 후 자동으로 URL이 생성돼요. 이 링크를 학생에게 공유하면 끝.
방법 3: GitHub Pages

GitHub에 파일을 올리면 무료로 웹사이트가 만들어져요. 여러 활동을 한 사이트에 모아서 관리할 때 좋아요. 다만 GitHub 사용법을 조금 알아야 해요.

처음이라면 방법 1(Antigravity 내장 배포) 또는 방법 2(Netlify 드래그 앤 드롭)를 추천해요. 둘 다 무료예요.
QR
QR코드 만들기

링크가 생겼으면 QR코드를 만들어서 교실에서 보여주세요. 학생이 스마트폰 카메라로 찍으면 바로 접속돼요.

1QR코드 생성 사이트 접속
구글에서 "QR코드 생성"을 검색하면 무료 사이트가 많아요.
2URL 붙여넣기
배포된 앱의 URL을 넣고 생성 버튼을 누르세요.
3교실에서 보여주기
QR코드 이미지를 PPT에 넣거나 프린트해서 교실에 붙이세요.
CLASSROOM
교실에서 활용하는 팁
수업 중 활용

도입 단계: 전 시간 배운 내용 복습용으로 3~5분 퀴즈를 풀게 하세요. 워밍업으로 딱이에요.

연습 단계: 새 문법을 설명한 뒤, 바로 앱으로 연습하게 하세요. "지금 배운 거 연습해볼까요?" + QR코드.

마무리 단계: 수업 끝나기 5분 전에 "오늘 배운 거 확인해볼게요" + 앱 링크.

수업 외 활용

숙제: 링크를 메시지로 보내주세요. "오늘 배운 에/에서, 집에서 한 번 더 풀어보세요."

자율 학습: 여러 활동 링크를 한 페이지에 모아두면 학생이 원하는 걸 골라서 풀 수 있어요.

학생들의 스마트폰 기종이 다양하니까, 배포 후 안드로이드와 아이폰 둘 다에서 테스트해 보세요.
