비교·변화 표현은 비교 대상이 눈에 보여야 자연스럽게 유도되거든요. "서울이 부산보다 인구가 많다"를 텍스트로만 연습하면 문형 연습에 그치지만, 실제 막대그래프를 보면서 말하면 데이터에서 의미를 읽어내고 그것을 언어로 전환하는 과정이 연습돼요.

그래프는 시각적으로 크기 차이, 순위, 증감 추세를 보여줘요. 학습자는 그래프를 보는 것만으로 "무엇이 더 크다", "무엇이 줄었다"를 인식하고, 남은 과제는 그것을 한국어로 표현하는 거예요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
그래프 수치가 자연스럽고 현실적인지 확인하세요. AI가 억지스러운 숫자를 쓸 수 있어요. 또한 질문이 실제로 목표 표현(-보다, -만큼, 증가/감소)을 유도하는지 확인하고, 모범답안의 문법이 TOPIK 3~4급 수준인지 점검해야 해요.
prompt.txt
# 역할 한국어 교육 전문가 # 과제 TOPIK 3~4급 학습자를 위한 그래프 읽기 활동 생성 # 조건 - 그래프 3개: 막대그래프 1개, 원그래프 1개, 꺾은선그래프 1개 - 각 그래프에 질문 3개씩 (총 9문항) - 목표 표현: -보다(비교), -만큼(동등), 가장(최상), 증가/감소 - 주제: 한국 생활 관련 (좋아하는 계절, 월별 관광객 수, 교통수단) - 각 문항의 모범답안 포함 # 출력 JSON 형식
data 필드의 수치를 코드 생성 도구에 넣으면 실제 그래프가 자동으로 그려져요. "위 데이터로 한국어 레이블의 막대그래프를 그려 달라"고 요청하면 돼요.
🔁 이렇게 바꿔서 응용하세요

그래프 읽기 대신 다른 의미 구성 활동으로 만들고 싶으면, 과제와 조건만 수정하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
비교 표현 대신 추측·원인 표현(-때문에, -는데)으로 바꿀 수 있어요.
조건
난이도를 조절하세요. 자유 서술→빈칸 채우기로 바꾸면 쉬워져요.
출력
앱이면 JSON, 인쇄용이면 표 형식으로 요청하면 돼요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 3개 그래프와 각각 3문항을 만들어 줘요. 파란색 키에 마우스를 올려보세요.

output.json
{ "charts": [ { "id": 1, "type": "bar", "title": "한국인이 선호하는 계절", "data": { "봄": 35, "여름": 15, "가을": 40, "겨울": 10 }, "unit": "%", "questions": [ { "q": "가장 선호하는 계절은 무엇인가요?", "model_answer": "가을이 가장 인기 있어요.", "target_grammar": "가장 (최상급)" }, { "...": "..." } ] } ] }
type
→그래프 유형이에요. bar(막대), pie(원), line(꺾은선) 중 하나예요.
data
→그래프에 들어갈 수치 데이터예요. 아래에서 코드로 자동 그려져요.
questions
→그래프를 보고 답할 질문과 모범답안이에요.
target_grammar
→해당 문항이 연습시키는 목표 표현이에요.
⚠️ AI가 만든 문항은 꼭 확인하세요. 어색한 표현이나 수준에 맞지 않는 어휘가 섞여 있을 수 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
🛠 Step 3. 코드 생성 도구로 앱을 만드세요

JSON 데이터가 준비됐으면, Antigravity 같은 코드 생성 도구에 붙여넣고 이렇게 말하면 돼요.

"이 JSON 데이터로 한국어 학습용 그래프 읽기 웹앱을 만들어줘. 각 그래프 아래에 질문 3개씩, 학습자가 텍스트로 답하게 해줘. 정답 제출하면 모범답안과 비교해서 맞으면 O, 틀리면 틀렸다고 알려줘."

완성되면 아래처럼 작동하는 앱이 나와요. 직접 풀어보세요!
