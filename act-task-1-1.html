설계

설계 근거

약국에서 약을 사는 건 한국 생활에서 학습자가 실제로 부딪히는 상황이에요. 교실에서 미리 연습해 두면 실제 상황에서의 불안이 줄어들거든요. 역할극은 실제 상황을 교실로 가져오는 가장 직접적인 방법이에요.
약국 상황에서는 증상 묘사, 정보 요청, 확인이 자연스럽게 결합돼요. 하나의 기능만 따로 연습하면 이 결합이 경험되지 않잖아요. 역할극은 여러 기능이 하나의 상황 안에서 자연스럽게 이어지는 경험을 제공해요.
핵심 설계 원칙: 환자 카드에 "두통"이라는 키워드만 주고 "머리가 아파요"라는 완성 문장을 주지 않는 게 중요해요. 학습자가 자신의 언어로 표현하도록 유도하는 거거든요.
유용 표현 카드의 역할: 이건 "비계(scaffolding)"예요. 초급에게는 역할극 전에 미리 제공하고, 중급에게는 역할극 후 확인용으로 활용하면 돼요.
생성

프롬프트 구성

AI로 역할 카드 세트를 생성하기 위한 체계적인 프롬프트입니다.

프롬프트 내용
pharmacy-roleplay.txt
역할: 한국어 교육 전문가 과제: TOPIK 2~3급 학습자를 위한 약국 역할극 카드 세트 생성 조건: - 3세트 난이도 순서: 세트1: 단순 (두통) 세트2: 복합 (기침+콧물+열) 세트3: 복합+알레르기 언급 - 각 세트마다 3장: ① 환자 카드 (증상 키워드만, 완성 문장 아님) ② 약사 카드 (약 정보) ③ 유용 표현 카드 - 환자 카드에는 키워드만 제시 (예: "두통" O, "머리가 아파요" X) 출력 형식: JSON { "sets": [ { "id": 1, "difficulty": "쉬움", "patient_card": { "symptoms": ["두통"], "duration": "오늘 아침부터", "allergies": "없음", "goal": "두통약을 사세요." }, "pharmacist_card": { "recommended": [ { "name": "약물명", "dosage": "복용량", "frequency": "빈도", "timing": "복용 시간", "price": "가격" } ], "questions_to_ask": ["약사가 물어볼 질문 1", "질문 2"] }, "useful_expressions": { "patient": ["환자 표현 1", "표현 2", ...], "pharmacist": ["약사 표현 1", "표현 2", ...] } }, ... ] }
JSON 필드 설명
sets
→
역할 카드 세트 배열 (3개)
patient_card
→
환자 카드 - 증상 키워드만 포함 (완성 문장 제외)
pharmacist_card
→
약사 카드 - 약 정보와 질문 리스트
useful_expressions
→
유용 표현 - 환자용/약사용 나누어 제시
recommended
→
추천 약 정보 (약물명, 복용량, 빈도, 타이밍, 가격)
이 프롬프트를 ChatGPT, Claude, Gemini 등 대형 언어 모델에 입력하면 JSON 형식으로 즉시 3세트의 역할극 카드를 생성할 수 있습니다.

필수 확인 항목
  • 3개 세트가 난이도 순서대로 구성됐는가? (단순 → 복합 → 복합+알레르기)

  • 유용 표현이 TOPIK 2~3급 수준에 맞는가?
  • 약사 카드의 질문이 자연스러운 회화 흐름을 유도하는가?

확장

활동 확장 방안

1단계: 사전 활동
역할극 전에 학습자들과 함께 실제 한국 약국의 특징을 논의하세요. 약사와의 상호작용이 의사와는 다른 점, 약국에서 구매할 수 있는 약의 종류 등을 사전 지식으로 쌓으면 역할극이 더욱 몰입감 있어질 거든요.
2단계: 역할극 수행
학습자 2명이 한 세트를 맡아 최소 2~3분의 대화를 진행하도록 하세요. 처음에는 카드를 보며 진행해도 괜찮아요. 더 높은 난이도라면 카드를 안 보고 자유 회화로 전환하는 것도 좋습니다.
3단계: 피드백
역할극을 진행한 후 발음, 자연스러움, 의사소통 성공 여부를 중심으로 피드백을 제시하세요. 부자연스러운 부분이 있다면 유용 표현 카드를 다시 함께 검토하는 것도 좋은 방법입니다.
4단계: 확장 활동
쓰기
역할극에서 나눈 대화를 글로 작성하고, 약사가 추천한 약의 복용법을 메모 형태로 정리
듣기
원어민 약사의 음성 샘플을 들으며 자연스러운 발음과 속도 확인
읽기
실제 약국 약설명서나 약 패키지의 한글 정보 읽기
상황 확장
약국 방문 후 집에서 약사의 조언을 실생활 일기로 기록
이 활동은 순수 역할극에 그치지 않고, 약국 방문이라는 실제 과제와 연결될 때 가장 큰 학습 효과를 거둬요. 역할극 후 학습자들이 실제로 약국을 방문하도록 권장하고, 그 경험을 되돌아보는 시간을 가지세요.
팁

효과적인 활용 방법

학습자 쌍 배치: 초급 학습자가 환자 역, 중급 학습자가 약사 역을 맡으면 난이도 차이를 보완할 수 있어요. 약사 역이 더 많은 언어 입력을 제공하기 때문입니다.
반복 실습: 같은 세트를 두 번 수행하게 하세요. 첫 번째는 카드를 보며, 두 번째는 카드 없이. 두 번째에는 더욱 자연스럽고 자신감 있는 표현이 나올 거예요.
문법 교정의 타이밍: 역할극 중간에 문법을 바로 잡지 마세요. 의사소통 흐름을 방해하거든요. 역할극이 끝난 후 피드백 시간에 주요 오류만 선택해서 다루는 것이 좋습니다.
