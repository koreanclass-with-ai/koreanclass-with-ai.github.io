직소(jigsaw)는 전체 정보를 여러 조각으로 나누어 각 학습자에게 하나씩 배분하는 구조예요. 자기가 읽은 내용을 다른 학습자에게 설명해야 하니까, 읽기와 말하기가 자연스럽게 연결되거든요.

읽기만 하면 수동적이지만, 읽은 내용을 남에게 설명해야 하면 능동적 처리가 필요해요. "내가 이해한 것을 전달할 수 있는가?"가 점검 기준이 되는 거죠.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
세 텍스트가 겹치지 않으면서도 보완적이어야 해요. 종합 질문을 풀려면 세 텍스트를 모두 읽어야 하는 구조가 되어야 거든요. 프롬프트에서 각 텍스트의 역할을 명확히 해 주세요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 3~4급 직소 읽기 자료 생성 # 조건 (Constraints) - 전체 주제: 한국의 사계절과 대표 음식 - 3분할 배치 - ∘ A: 봄과 여름 계절 특징과 음식 - ∘ B: 가을과 겨울 계절 특징과 음식 - ∘ C: 사계절을 통한 한국 음식 문화의 변화 - 각 텍스트: 150~200자, TOPIK 3급 어휘 수준 - 종합 질문 5개 (세 텍스트 모두에서 정보 필요) # 출력 (Output) JSON 형식
직소는 텍스트의 역할 분담이 핵심이에요. 각 텍스트가 "자신의 부분 정보만" 담기 때문에, 전체 상을 파악하려면 세 텍스트를 모두 읽어야 하거든요.
🔁 이렇게 바꿔서 응용하세요

주제만 바꾸면 어떤 내용이든 직소 활동으로 만들 수 있어요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
주제를 바꾸세요. "서울의 지역별 특징", "한국 명절의 의미와 음식" 등.
조건
인원을 조정하세요. 2인 활동이면 텍스트 2개, 4인이면 4개로 분할하면 돼요.
출력
인쇄할 때는 "A4 워크시트 형식"으로 바꾸세요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 텍스트와 질문을 만들어 줘요. 파란색 키에 마우스를 올려보세요.

output.json
{ "topic": "한국의 사계절과 대표 음식", "texts": [ { "role": "A", "subtitle": "봄과 여름", "text": "봄이 되면... }, { "role": "B", "...": "..." } ], "synthesis_questions": [ { "q": "계절에 따라 한국 음식이 어떻게 변하나요?", "required_texts": ["A", "B", "C"] }, { "q": "...", "required_texts": ["..."] } ] }
texts
→3개의 텍스트 조각이 들어가요. 각 조각은 역할(A/B/C)과 내용을 가져요.
synthesis_questions
→세 텍스트를 모두 읽어야 답할 수 있는 종합 질문이에요.
required_texts
→각 질문에 필요한 텍스트 목록이에요. [A, B, C] 세 개 모두 필요하면 활동이 제대로 설계된 거예요.
⚠️ 종합 질문이 정말 세 텍스트를 모두 필요로 하는지 꼭 확인하세요. 한 두 텍스트만으로 답할 수 있으면 직소 효과가 떨어져요.
🖨 Step 3. 인쇄용 자료로 정리하세요

JSON이 준비됐으면, 아래처럼 프롬프트를 추가해서 인쇄 가능한 형식으로 변환하면 돼요.

"이 데이터를 A4 워크시트로 정리해줘. 3개의 텍스트는 3개 섹션으로 나누고, 아래에 종합 질문 5개와 답쓰는 칸을 추가해줘."

봄과 여름의 한국 음식

봄이 되면 새로운 채소와 산나물이 나와요. 이때 사람들은 쑥개떡이나 봄나물 무침을 많이 먹어요. 여름이 오면 더운 날씨 때문에 시원한 음식이 인기가 있어요. 냉면, 콩국수, 빙수 같은 음식들이 여름의 대표 음식이에요.

1. 계절에 따라 한국 음식이 어떻게 변하나요?
2. 봄과 여름의 차이는 뭐가 있나요?

가을과 겨울의 한국 음식

가을이 오면 수확의 계절이 돼요. 밤, 대추, 곡식들이 많이 나오고, 송편이나 대추차 같은 음식들이 만들어져요. 겨울은 가장 추운 계절이라 따뜻한 음식이 필요해요. 떡국, 팥죽, 김장 김치 같은 음식들이 겨울을 보내는 데 도움이 돼요.

1. 가을과 겨울의 음식 차이는 무엇인가요?
2. 왜 겨울에는 따뜻한 음식을 먹을까요?

계절을 통한 한국 음식 문화

한국 음식은 사계절의 변화를 중심으로 발전해 왔어요. 각 계절에 나는 재료들을 최대한 활용하고, 그 계절의 기후에 맞는 음식을 만들어 왔거든요. 이런 방식은 건강하고 자연스러운 식생활을 가능하게 했어요.
