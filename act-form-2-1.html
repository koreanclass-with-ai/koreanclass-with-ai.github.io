INTRO
💡 왜 이 형식으로 만들었나요?
활동 유형대입 연습 (카드 기반)
범주❶ 형태 학습
대상초급 (TOPIK 1급)
목표'N은/는 N이에요/예요' 구문 반복 연습
언어 기능말하기
규모12장 카드 / 5–7분
배포 형태인쇄용 카드 세트 (디지털 플래시카드 변형 가능)

대입 연습은 같은 구문을 여러 번 반복하는 활동이에요. 워크시트에 문장 10개를 쭉 나열하면 학습자가 기계적으로 쓰게 되잖아요. 그런데 카드를 무작위로 뽑는 형식이면 "다음에 뭐가 나올까"라는 기대가 생기고, 같은 반복이라도 지루함이 줄어들거든요.

왜 대입 연습이 'N이에요/예요'를 가르치기에 적합한가요? '이에요/예요'의 습득에서 핵심은 구문 구조("저는 ___이에요/예요")를 자동화하는 거예요. 구문 자동화는 같은 틀 안에서 다양한 어휘를 반복적으로 넣어보는 경험에서 나온다고 할 수 있어요. 대입 연습은 바로 이 "틀은 고정, 내용만 교체"를 반복시키는 형식이에요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) 직업 어휘 대입 연습용 카드 자료 생성 # 조건 (Constraints) - 직업 12개 (초급 수준, 학습자가 자주 접하는 직업) - 각 직업: 직업명(한국어), 영어 번역, 설명(한국어 1문장) - 받침 유무가 섞이도록: - 받침 있는 직업 6개 (→ 이에요) - 받침 없는 직업 6개 (→ 예요) # 출력 (Output) JSON 형식
카드 일러스트는 이미지 생성 도구로 만들 수 있어요. "simple flat illustration of a teacher, white background, minimal style" 같은 프롬프트가 깔끔한 결과를 내요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 카드 데이터를 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "cards": [ { "id": 1, "job_kr": "선생님", "job_en": "teacher", "description": "학교에서 학생을 가르치는 사람", "has_batchim": true, "example": "저는 선생님이에요." }, { "id": 2, "job_kr": "가수", "job_en": "singer", "description": "노래를 부르는 사람", "has_batchim": false, "example": "저는 가수예요." }, { "id": 3, "...": "..." } ] }
job_kr
→한국어 직업명이에요. 카드 앞면에 표시돼요.
job_en
→영어 번역이에요. 초급 학습자의 이해를 도와요.
has_batchim
→받침 유무예요. true면 '이에요', false면 '예요'가 붙어요.
example
→완성된 예시 문장이에요. 카드 뒷면에 넣어요.
description
→직업을 설명하는 한 문장이에요.
⚠️ AI가 만든 카드 데이터는 꼭 확인하세요. 어색한 표현이나 수준에 맞지 않는 어휘가 섞여 있을 수 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
🔁 응용 레이어

이 프롬프트의 각 부분을 바꾸면 다양한 카드 자료를 만들 수 있어요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
어휘장만 바꾸세요. "음식 어휘 대입 연습용 카드", "장소 어휘 카드" 등으로요.
조건
레벨과 특성에 맞게 조정하세요. 중급이면 구문도 복잡하게: 'N을/를 좋아해요', 'N에서 왔어요' 등.
출력
앱으로 만들 거면 JSON, 인쇄용이면 "카드 앞면/뒷면 표 형식으로 정리해줘"로 바꾸세요.
🛠 Step 3. 카드를 인쇄하거나 앱으로 만드세요

JSON 데이터가 준비됐으면, 두 가지 방법이 있어요.

방법 1. 인쇄용 카드: AI한테 "이 JSON을 카드 앞면/뒷면으로 정리해서 A4 엑셀 표로 만들어줘"라고 하면 돼요. 그러면 인쇄해서 재단할 수 있어요.

방법 2. 디지털 플래시카드: Antigravity 같은 코드 생성 도구에 데이터와 "이 JSON으로 플래시카드 웹앱 만들어줘. 카드를 클릭하면 뒤집어지고, 앞면에 직업명, 뒷면에 예시 문장 나와야 해"라고 하면 앱이 나와요.

바꾸고 싶은 것방법예시

구문 교체'N이에요/예요' → 다른 구문'N을/를 좋아해요', 'N에 가요', 'N에서 왔어요'
