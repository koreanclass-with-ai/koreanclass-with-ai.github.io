불규칙 활용의 핵심은 "기본형에서 활용형을 만들어내는 과정"이거든요. "듣다 + -아요/어요 = ?"라는 과제에서 학습자는 불규칙 규칙을 스스로 적용해야 해요.

변환 연습이 불규칙 활용을 가르치기에 적합한 이유는요, 불규칙 활용의 학습 목표는 "이 동사가 규칙인지 불규칙인지 판단하고, 불규칙이면 해당 규칙을 적용하는 것"이거든요. 규칙 활용 문항(먹다→먹어요)과 불규칙 문항(듣다→들어요)을 나란히 배치하면, 학습자가 매 문항에서 능동적으로 판단하게 돼요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
규칙 활용 문항(먹다→먹어요)의 오답 해설에서는 "이 동사는 규칙 활용입니다"라고 명시해주세요. 학습자가 "이건 불규칙이 아니구나"를 확인하는 것 자체가 학습이거든요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 2급 학습자를 위한 불규칙 활용 연습 문항 생성 # 조건 (Constraints) - 문항 수: 12개 - 불규칙 유형별 3개씩: ㄷ불규칙(듣다, 걷다, 묻다), ㅂ불규칙(덥다, 춥다, 쉽다), ㅎ불규칙(빨갛다, 노랗다, 하얀다) - 규칙 활용 대비용 3개(먹다, 읽다, 받다) - 각 문항: 기본형 + 결합 어미(-아요/어요) → 정답(활용형) - 오답 해설에서 불규칙 규칙 명시 # 출력 (Output) JSON 형식
역할과 출력 형식은 그대로 두고, 과제와 조건만 바꾸면 어떤 불규칙 유형이든 만들 수 있어요.
🔁 이렇게 바꿔서 응용하세요

불규칙 유형을 바꾸고 싶으면, 과제와 조건만 수정하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
불규칙 유형을 바꾸세요. "ㅅ불규칙(낫다), 르불규칙(모르다)" 등을 추가할 수 있어요.
조건
어미를 다양화하세요. -아서/어서, -(으)면, -(으)ㄴ 등 다른 어미도 적용할 수 있어요.
출력
앱으로 만들 거면 JSON, 인쇄용이면 "표 형식으로 정리해줘"로 바꾸세요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 문항을 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "questions": [ { "id": 1, "base_form": "듣다", "ending": "-아요/어요", "answer": "들어요", "type": "ㄷ불규칙", "rule": "모음 어미 앞에서 ㄷ→ㄹ" }, { "id": 2, "base_form": "먹다", "ending": "-아요/어요", "answer": "먹어요", "type": "규칙", "rule": "규칙 활용 — 받침 그대로 유지" }, { "id": 3, "...": "..." } ] }
base_form
→동사/형용사의 기본형이에요. 학습자에게 제시돼요.
ending
→결합할 어미예요. -아요/어요가 기본이에요.
answer
→정답 활용형이에요.
type
→불규칙 유형이에요. ㄷ/ㅂ/ㅎ불규칙 또는 규칙.
rule
→적용되는 규칙 설명이에요. 해설에 표시돼요.
⚠️ AI가 만든 문항은 꼭 확인하세요. 어색한 표현이나 수준에 맞지 않는 어휘가 섞여 있을 수 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
🛠 Step 3. 코드 생성 도구로 앱을 만드세요

JSON 데이터가 준비됐으면, Antigravity 같은 코드 생성 도구에 붙여넣고 이렇게 말하면 돼요.

"이 데이터로 한국어 학습용 불규칙 활용 연습 웹앱 만들어줘. 기본형 + 어미를 입력하면 즉시 정답/오답 표시되고, 규칙/불규칙 해설 나오고, 12문항 다 풀면 점수 보여줘."

완성되면 아래처럼 작동하는 앱이 나와요. 직접 눌러보세요!
