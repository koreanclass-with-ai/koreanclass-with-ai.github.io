읽기 활동에서 가장 흔한 문제가 뭐냐면, 텍스트가 학습자 수준에 안 맞는 거예요. 너무 어려우면 사전 찾기에 시간을 다 쓰고, 너무 쉬우면 도전이 없잖아요. AI 도구를 쓰면 같은 주제를 초급/중급/고급 세 버전으로 만들 수 있어서, 같은 교실에서도 수준별 자료를 제공할 수 있어요.

읽기 텍스트는 학습자에게 풍부한 맥락을 제공해요. 목표 문법이 실제 맥락에서 어떻게 사용되는지를 경험하고, 이해 확인 문항으로 자신의 이해를 점검하는 거죠.

BUILD
🤖 Step 1. AI한테 레벨별 텍스트를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 레벨별 텍스트를 만들 때 주의하세요
각 레벨의 어휘와 문법이 명확하게 다르지 않으면, 학습자들이 "이게 뭐가 다른 텍스트지?" 하며 혼란스러워해요. 프롬프트에 "TOPIK 2급 어휘만" 이런 식으로 자세히 명시해 줄수록 AI가 정확한 결과를 내놓아요.
prompt.txt
# 역할: 한국어 교육 전문가 # 과제: "한국의 사계절"을 주제로 레벨별 읽기 텍스트 + 이해 확인 문항 생성 # 조건: - 레벨 3단계: 초급(TOPIK 1~2급), 중급(TOPIK 3~4급), 고급(TOPIK 5~6급) - 초급: 4~5문장, TOPIK 1급 어휘, 단문 위주 - 중급: 8~10문장, TOPIK 3급 어휘, 접속문·내포문 포함, 비교 표현 사용 - 고급: 12~15문장, TOPIK 5급 어휘, 복문·피동·사동, 문화적 맥락 포함 - 각 레벨에 이해 확인 문항 5개 (4지선다) # 출력: JSON 형식
어휘 수준을 통제하려면 "TOPIK 2급 어휘 목록에 있는 단어만 사용해 달라"고 명시하세요. 다만 AI가 이 목록을 완벽하게 기억하지는 않으니까, 출력 후 교사가 어휘를 점검해야 해요.
🔁 이렇게 바꿔서 응용하세요

주제와 레벨을 바꿔서 다양한 읽기 자료를 만들 수 있어요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 주제에도 통해요.
과제
주제를 바꾸세요. 한국 음식, 교통, K-pop, 명절 등 뭐든 가능해요.
조건
문항 유형을 서술형으로 바꾸거나, 듣기 활동으로 전환 가능해요. TTS 도구랑 연결하면 되죠.
출력
웹 앱이면 JSON, 인쇄하려면 "표 형식"으로 요청하세요.
📤 Step 2. 결과를 확인하고 점검하세요

AI가 아래처럼 JSON으로 세 레벨의 텍스트를 만들어줄 거예요. 각 레벨을 읽어보고, 어휘와 문법이 실제로 다른지 확인하는 게 중요해요.

output.json
{ "topic": "한국의 사계절", "levels": [ { "level": "초급 (TOPIK 1~2급)", "text": "한국에는 사계절이 있어요...", "word_count": 54, "questions": [ { "q": "한국에 계절이 몇 개 있어요?", "choices": ["2개", "3개", "4개", "5개"], "answer": "4개" }, { "...": "..." } ] }, { "level": "중급 (TOPIK 3~4급)", "...": "..." }, { "level": "고급 (TOPIK 5~6급)", "...": "..." } ] }
level
→텍스트의 수준이에요. 초급/중급/고급 중 하나예요.
text
→읽기 텍스트 본문이에요. 같은 주제를 수준에 맞게 표현한 거죠.
word_count
→글자 수를 세어놨어요. 초급은 짧고, 고급은 길어야 맞아요.
questions
→이해 확인 문항 배열이에요. 4지선다 5개가 들어가요.
answer
→정답을 넣는 곳이에요. choices 배열에 들어 있는 선택지 중 하나예요.
⚠️ AI가 만든 텍스트는 반드시 읽어보세요. 그럴듯해 보이지만 부자연스러운 표현이 섞여 있을 수 있어요. 특히 고급 텍스트가 정말 고급 수준인지 확인하세요. AI는 준비된 초안을 주는 조수고, 최종 판단은 선생님 몫이에요.
🛠 Step 3. 코드 생성 도구로 앱을 만드세요

JSON 데이터가 준비됐으면, Antigravity 같은 코드 생성 도구에 붙여넣고 아래처럼 말하면 돼요.

"이 데이터로 한국어 학습용 수준별 읽기 앱 만들어줘. 텍스트 읽고 4지선다 문항 5개를 풀면, 정오표와 점수가 나와. 답을 고르면 즉시 피드백을 보여줘."

완성되면 아래처럼 작동하는 앱이 나와요. 직접 눌러보세요!
