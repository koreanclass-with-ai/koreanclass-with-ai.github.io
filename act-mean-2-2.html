듣기 자료 제작의 가장 큰 병목은 녹음이에요. 원어민 화자를 섭외하고, 녹음하고, 편집하는 데 시간과 비용이 엄청 들잖아요. AI 음성 생성 도구를 쓰면 대본만 있으면 즉시 음성이 만들어지고, 속도·화자·톤을 자유롭게 조절할 수 있어요.

대화를 듣고 핵심 정보(시간, 장소, 인물, 목적)를 파악하는 과정은 의미에 집중하게 해요. "이 문법이 뭐지?"가 아니라 "이 사람이 어디에 가려는 거지?"를 생각하면서 듣는 거예요.

BUILD
🤖 Step 1. AI한테 대화 스크립트를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
대화가 자연스러운 일상 대화인지, 각 문항이 묻는 정보가 대화에 명확히 포함되어 있는지 확인해야 해요. AI가 만든 스크립트는 꼭 확인하고 수정해서 써야 돼요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 2~3급 듣기 대화 스크립트 + 이해 문항 생성 # 조건 (Constraints) - 대화 3개 / 각 6~8턴 - 주제: 약속 정하기 · 쇼핑 · 길 묻기 - 각 대화에 이해 문항 3개 (4지선다) - 묻는 정보: 시간, 장소, 가격, 방향 등 핵심 정보 - 어휘: TOPIK 2~3급 수준 (초중급) # 출력 (Output) JSON 형식
대화의 각 턴을 음성 생성 도구에 개별적으로 입력하면 화자별 다른 목소리를 설정할 수 있어요. 민수는 남성 음성, 지영은 여성 음성으로 설정하면 화자 구분이 자연스러워요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 대화와 문항을 만들어 줘요. 파란색 키에 마우스를 올려보세요.

output.json
{ "dialogues": [ { "id": 1, "topic": "약속 정하기", "speakers": ["민수", "지영"], "script": [ { "speaker": "민수", "text": "토요일에 시간 있어요?" }, { "speaker": "지영", "text": "네, 오후에요. 왜요?" } ], "questions": [ { "q": "두 사람은 무엇을 할 거예요?", "choices": ["영화 보기", "쇼핑", "공부", "운동"], "answer": "영화 보기" }, { "q": "...", "choices": ["..."], "answer": "..." } ] } ] }
topic
→대화의 주제를 설명해요. 약속 정하기, 쇼핑, 길 묻기 등.
speakers
→대화에 참여하는 화자들의 이름이에요.
script
→대화의 각 턴(speaker + text)을 순서대로 담고 있어요.
questions
→대화 이해를 확인하는 4지선다 문항들이에요.
answer
→각 문항의 정답을 나타내요.
⚠️ AI가 만든 대화는 꼭 확인하세요. 어색한 표현이나 수준에 맞지 않는 어휘가 섞여 있을 수 있어요.
🎙️ Step 3. TTS 도구로 음성을 생성하세요

Google Translate, NAVER Papago, Azure Text-to-Speech 같은 무료 TTS 도구를 이용해요. 스크립트의 각 턴을 복사해서 붙여넣고 음성 파일을 저장하면 돼요.

팁: 민수의 대사는 남성 음성으로, 지영의 대사는 여성 음성으로 설정하면 학습자가 화자를 자연스럽게 구분할 수 있어요.
