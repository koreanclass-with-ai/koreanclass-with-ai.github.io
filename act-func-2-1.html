길 안내하기

AI로 지도 기반 방향 설명 자료 만드는 법

프롬프트로 지도 기반 길 안내 활동 자료를 만드는 과정을 안내해요.

TOPIK 2급 · 초중급 제작 시간 10분 지도 2개 × 경로 2개

설계 근거

길 안내는 방향, 위치, 거리를 말로 전달하는 활동이에요. 지도는 이 세 가지를 시각적으로 제공하니까, 학습자는 보이는 것을 말로 옮기기만 하면 돼요. 내용(어디로 가는가)은 지도가 제공하고, 형식(어떻게 말하는가)은 학습자가 선택하는 거예요.

길 안내는 순서가 중요하잖아요. "먼저 직진하세요, 그다음에 오른쪽으로 가세요"처럼 단계별로 설명해야 하거든요. 지도 위에 경로가 표시되어 있으면 학습자가 순서를 놓치지 않고 단계별로 설명할 수 있어요.

프롬프트 구성

프롬프트

# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 2급 학습자를 위한 길 안내 활동 자료 생성 # 조건 (Constraints) - 지도 2개 (동네 지도 스타일, 단순화된 격자형) - 각 지도에 경로 2개 (쉬운 경로 + 어려운 경로) - 목표 표현: 직진하다, 오른쪽/왼쪽으로 돌다, ~이/가 있다, ~옆에/앞에/뒤에, -(으)면 ~이/가 보이다 - 건물: 은행, 편의점, 학교, 병원, 카페 등 - 각 경로: 모범 설명문(3~5단계), 핵심 표현 목록 # 출력 (Output) JSON 형식

응답 구조 (JSON)

{ "maps": [ { "id": 1, "landmarks": [ {"name": "지하철역", "position": "A1"}, {"name": "은행", "position": "B2"}, {"name": "카페", "position": "C3"} ], "routes": [ { "from": "지하철역", "to": "카페", "difficulty": "쉬움", "steps": [ "지하철역에서 나오세요.", "오른쪽으로 가세요.", "직진하면 은행이 보여요.", "은행 옆에 카페가 있어요." ] } ] } ] }

응답 구조 설명

키 설명
maps 지도 목록이에요. 여러 지도의 정보를 담아요.
landmarks 지도 위의 건물이에요. 이름과 위치가 들어가요.
routes 경로 정보예요. 출발~도착, 난이도, 단계별 설명이 있어요.
steps 단계별 길 안내 설명문이에요. 각 단계는 한 문장씩 구성돼요.
difficulty 경로의 난이도예요. '쉬움' 또는 '어려움'으로 나뉘어요.

경로 1 (쉬움): 지하철역 → 카페

  1. 지하철역에서 나오세요.
  2. 앞으로 직진하세요.
  3. 직진하면 은행이 보여요.
  4. 은행 옆에 카페가 있어요.

핵심 표현: 직진하다 · ~이/가 있다 · ~옆에 · -(으)면 ~이/가 보이다

경로 2 (어려움): 지하철역 → 학교

  1. 지하철역에서 나오세요.
  2. 앞으로 직진하세요.
  3. 병원이 보이면 오른쪽으로 꺾으세요.
  4. 다시 직진하세요.
  5. 왼쪽에 학교가 있어요.

핵심 표현: 직진하다 · 오른쪽으로 꺾다 · 왼쪽 · -(으)면 · ~이/가 있다

활동 확장 방안

난이도 ↑ 경로를 복잡하게 하기 (회전 2회 이상, 교차로 통과) / 건물 수 늘리기 / 거리 표현 추가 ("100미터 가세요")
인터랙션 개인 활동 → 정보 차 짝 활동 (A만 지도를 보고 B에게 구두로 안내 / 그다음 역할 바꾸기)
언어 기능 말하기 → 쓰기 (길 안내 이메일 작성) / 말하기 + 듣기 (음성 녹음 후 학습자가 지도에 따라 움직이기)
과제 AI를 사용해 다양한 지도 스타일 생성 / 학습자가 직접 프롬프트 수정해 새로운 경로 만들기
