설득은 일방적 주장과 달라요. 상대방이 반대 의견을 가지고 있으니까, 자기 주장만 반복해서는 설득할 수 없거든요. 상대의 논거를 듣고, 인정하고, 그에 대한 반론을 제시하는 과정이 필요해요. 시나리오에 상대방의 반대 이유가 명시되어 있으면 학습자가 "이 반론에 어떻게 대응할까"를 준비할 수 있어요.

설득은 실제 대화 안에서 일어나잖아요. "당신은 A입장이고, 상대는 B입장입니다. 상대를 설득하세요"라는 시나리오는 설득의 핵심 — 상대의 관점을 이해하고 그에 대응하는 것 — 을 자연스럽게 연습시켜 줘요.

B카드에 "양보 가능한 조건"을 넣는 이유는, 대화가 평행선만 달리지 않고 합의점을 찾을 수 있게 하기 위해서예요. 설득 활동의 성공은 "상대를 완전히 이기는 것"이 아니라 "절충안을 도출하는 것"이거든요.
BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
A카드와 B카드의 근거가 실제로 대립하도록 설계해야 해요. A가 "효율성"을 주장하는데 B가 "비용"만 반대하면 논점이 어긋나서 설득 연습이 안 돼요. 같은 논점에서 찬반이 부딪혀야 해요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 5~6급 학습자를 위한 설득하기 시나리오 생성 # 조건 (Constraints) - 시나리오 3개: 1. 회사에서 원격 근무 도입 (부서장 설득) 2. 대학에서 교양 과목 필수화 (학생회 토론) 3. 우리 동네에 대형 마트 유치 (주민 회의) - 각 시나리오: A(설득자) 카드 + B(반대자) 카드 + 상황 카드 - A카드: 설득 목표, 핵심 근거 3개, 예상 반론 대응 힌트 - B카드: 반대 이유 3개, 양보 가능한 조건 1개 - 목표 표현: 물론 ~지만, -는 점에서, -다는 것을 인정하지만, 오히려, 그럼에도 불구하고 # 출력 (Output) JSON 형식
역할과 출력 형식은 그대로 두고, 과제와 조건만 바꾸면 어떤 설득 주제든 만들 수 있어요.
🔁 이렇게 바꿔서 응용하세요

다른 설득 상황으로 만들고 싶으면, 시나리오와 근거만 교체하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 의견 표현 활동에도 통해요.
과제
시나리오 주제를 교체하면 돼요. 환경, 교육, 기술, 사회 이슈 등 다양하게 가능해요.
조건
근거 수를 줄이면(2개) 쉬워지고, 늘리면(4~5개) 어려워져요. 양보 조건 유무로도 조절 가능해요.
출력
앱이면 JSON, 인쇄용이면 "A4 카드 형식으로 정리해줘"로 변경하면 돼요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 시나리오를 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "scenarios": [ { "id": 1, "situation": "회사에서 원격 근무 도입을 논의하는 부서 회의", "card_A": { "role": "원격 근무 찬성자", "goal": "부서장을 설득하여 주 2일 원격 근무를 도입하게 하세요.", "arguments": ["출퇴근 시간 절약", "집중력 향상", "직원 만족도"], "counter_hints": ["협업 부족 반론 → 화상회의 도구 활용 제안"] }, "card_B": { "role": "부서장 (반대)", "arguments": ["팀 협업 어려움", "근태 관리 불가", "보안 문제"], "compromise": "시범 운영 기간(1개월)을 두는 것에는 열려 있다" } }, { "id": 2, "...": "..." } ] }
scenarios
→설득 시나리오 목록이에요. 3개의 시나리오가 들어가요.
situation
→상황 설명이에요. 어떤 맥락에서 설득이 일어나는지 알려줘요.
card_A
→설득자 카드예요. 목표, 근거, 반론 대응 힌트가 있어요.
card_B
→반대자 카드예요. 반대 이유와 양보 가능 조건이 있어요.
counter_hints
→예상 반론 대응 힌트예요. 상대의 반론에 어떻게 대응할지 실마리를 줘요.
compromise
→양보 가능한 조건이에요. 합의점을 찾기 위한 단서가 돼요.
⚠️ AI가 만든 시나리오는 꼭 확인하세요. A와 B의 근거가 실제로 대립하는지, 양보 조건이 자연스러운지 점검이 필요해요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
