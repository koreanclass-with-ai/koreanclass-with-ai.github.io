조건문은 가정 상황을 제시하고 반응을 요구하는 구조예요. 카드에 "만약 ~면"의 조건을 인쇄하고 학습자가 응답을 완성하면, 자연스럽게 조건-결과의 언어 구조가 연습되거든요.

조건문은 "가정 → 반응"의 2단계 구조잖아요. 카드가 가정을 제공하고 학습자가 반응을 만드는 형식은 이 2단계를 물리적으로 분리시켜요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
모범 응답은 학습자가 참고할 기준 예시예요. 다양한 문법 형태(-(으)ㄹ 거예요, -고 싶어요, -아야 해요 등)를 포함하면 학습자가 자신의 응답과 비교할 때 더 풍부한 학습이 되어요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 2~3급 조건부 응답 카드 생성 # 조건 (Constraints) - 카드 8장 - 각 카드: 조건절 (-(으)면 또는 -ㄹ 때) + 모범 응답 2~3개 - 일상생활 상황 중심 (학교, 직장, 집, 여행 등) - 개방형 (다양한 응답 가능) - 결과 표현: -(으)ㄹ 거예요, -고 싶어요, -아야 해요 등 다양화 # 출력 (Output) JSON 형식
역할과 출력 형식은 그대로 두고, 과제와 조건만 바꾸면 어떤 문법 항목이든 만들 수 있어요.
🔁 이렇게 바꿔서 응용하세요

조건 표현을 바꾸고 싶으면, 과제와 조건만 수정하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 문법에도 통해요.
과제
조건 표현을 교체하세요. "-(으)ㄹ 때", "-거든", "-다면" 등 다양한 조건 표현 활용
조건
모범 응답 수를 조절하거나, 조건+결과를 모두 학습자가 작성하도록 변경하면 난이도 상향
출력
앱이면 JSON, 인쇄용이면 "카드 형식으로 정리해줘"로 변경
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 카드를 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "cards": [ { "id": 1, "condition": "비가 오면...", "condition_grammar": "-(으)면", "model_answers": [ "우산을 가져갈 거예요.", "집에서 쉬고 싶어요.", "카페에 가야 해요." ] }, { "id": 2, "...": "..." } ] }
condition
→조건절이에요. "만약 ~면"의 가정 부분이죠.
condition_grammar
→사용된 문법 형태예요. -(으)면, -ㄹ 때, -거든 등이에요.
model_answers
→모범 응답 2~3개를 담은 배열이에요. 다양한 결과 표현이 포함돼요.
⚠️ AI가 만든 카드는 꼭 확인하세요. 조건이 실현 가능한지, 모범 응답이 자연스러운지 검토하세요. AI는 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
