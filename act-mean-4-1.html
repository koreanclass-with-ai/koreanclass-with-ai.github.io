범주❷ 의미 구성
대상초중급 (TOPIK 2급)
목표질문과 답변을 통해 서로의 정보를 교환하여 표 완성
언어 기능말하기 · 듣기
규모A/B 카드 세트 1개 / 10–15분
배포인쇄용 카드 세트

정보 차는 학습자 간 상호작용을 가장 직접적으로 유발하는 구조예요. A가 아는 것을 B가 모르고, B가 아는 것을 A가 모르니까, 둘 다 서로에게 질문하고 답해야 해요. 교사가 시키지 않아도 자연스럽게 대화가 일어나거든요.

빈 표의 칸을 채우는 구체적 목표가 있으니까, 의사소통의 성공/실패가 눈에 보여요. 표가 다 채워졌는지 확인하면 되니까요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
정보 차는 A카드와 B카드가 정확히 상보적이어야 해요. A에만 있는 정보, B에만 있는 정보가 명확하게 나뉘어야 자연스럽게 질문이 일어나거든요. 프롬프트에서 정확히 명시해 줄 수록 좋아요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 2급 정보 차 카드 세트 생성 # 조건 (Constraints) - 주제: 반 친구 개인정보 (이름, 나라, 전화번호, 취미, 생일) - 인물 5명 - A카드: 이름+나라 있고 나머지 비어있음 - B카드: 전화번호+취미+생일 있고 나머지 비어있음 - 양방향 상호작용 가능하도록 배분 - 문형: "~이/가 뭐예요?", "~이/가 어디예요?", "~이/가 언제예요?" # 출력 (Output) JSON 형식
A카드와 B카드를 같은 레이아웃으로 만들되, 비어 있는 칸만 다르게 하면 학습자가 구조를 금방 파악해요.
🔁 이렇게 바꿔서 응용하세요

주제와 질문 문형만 바꾸면 어떤 정보 차든 만들 수 있어요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
주제를 바꾸세요. "가게 정보", "여행지", "시간표" 등.
조건
질문 문형을 특정 문법으로 제한하세요. "-을/를 좋아해요", "-어서 → 이유 표현" 등.
출력
인쇄할 때는 "표 형식으로 정리해줘"로 바꾸세요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 인물과 카드 정보를 만들어 줘요. 파란색 키에 마우스를 올려보세요.

output.json
{ "people": [ { "id": 1, "name": "민준", "country": "중국", "phone": "010-1234-5678", "hobby": "영화 보기", "birthday": "8월 15일" }, { "id": 2, "...": "..." } ], "card_A": { "visible": ["name", "country"], "hidden": ["phone", "hobby", "birthday"] }, "card_B": { "visible": ["phone", "hobby", "birthday"], "hidden": ["name", "country"] } }
people
→5명의 인물 정보 배열이에요.
card_A / card_B
→visible: 카드에 보이는 칸, hidden: 비어 있는 칸이에요.
상보성
→A의 hidden = B의 visible이어야 정보 차가 완성돼요.
⚠️ 정보 차는 상보성이 생명이에요. A와 B의 정보가 정확히 나누어지지 않으면 활동이 자연스럽지 않아요. JSON을 받으면 꼭 확인하세요.
🖨 Step 3. 인쇄용 카드로 변환하세요

JSON 데이터가 준비됐으면, 아래처럼 프롬프트를 추가해서 인쇄 가능한 카드 형식으로 변환하면 돼요.

"이 데이터를 A4 카드 2장으로 정리해줘. 한 장은 A카드, 한 장은 B카드. 각 행에 한 명씩, 비어있는 칸은 빈 줄(_____)로 표시해줘."

PRACTICE
🎯 완성된 자료의 모습

이 활동은 웹앱 데모 대신 완성된 자료의 모습을 보여줍니다.

A 카드
이름과 나라를 알고, 나머지를 묻는 카드
이름나라전화번호취미생일
민준중국?????????
영미일본?????????
B 카드
전화번호·취미·생일을 알고, 이름과 나라를 묻는 카드
이름나라전화번호취미생일
??????010-1234-5678영화 보기8월 15일

짝 활동에서 학습자가 "~이/가 뭐예요?"라고 자연스럽게 질문하도록, A와 B의 정보를 정확히 나누세요.
CHECK
✅ 이것만 확인하세요
  • A카드와 B카드의 정보가 정확히 상보적인가요? (A의 비어있는 것 = B의 있는 것)
  • 문형이 적절한가요? ("~이/가 뭐예요?", "~이/가 어디예요?", "~이/가 언제예요?" 사용 가능)
  • 정보가 충분히 다양한가요? (모두 같은 나라, 같은 취미는 좋지 않아요)
  • 인물 5명이 명확하게 구별되나요?
