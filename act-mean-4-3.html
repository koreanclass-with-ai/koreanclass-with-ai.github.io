서베이는 학습자에게 "실제로 사람에게 말을 걸어야 하는 이유"를 만들어줘요. 빈 표를 채워야 하니까 질문을 해야 하고, 답을 들어야 하고, 기록해야 해요. 이 과정이 말하기+듣기+쓰기를 자연스럽게 통합하거든요.

질문이 미리 활동지에 인쇄되어 있으니까, 초급 학습자도 질문을 읽고 따라 말하기만 하면 돼요. 안전한 말하기 환경이 되는 거죠.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
초급 학습자가 따라 말하기 쉽도록, 질문 문형을 제한해야 해요. "~이/가 뭐예요?", "~을/를 좋아해요?", "~에 가요?" 같은 익숙한 패턴 몇 개만 사용하도록 프롬프트에서 명시하세요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 1~2급 인터뷰 활동지 생성 # 조건 (Constraints) - 질문 5개 - 인터뷰 대상: 3명 - 문형: "~이/가 뭐예요?", "~을/를 좋아해요?", "~에 가요?" - 기록란: 이름 + 질문별 답변 칸 - 정리 발표: "가장 인기 있는 ~은/는 ~이에요" 형식 # 출력 (Output) JSON 형식
초급 학습자들은 텍스트된 질문을 읽고 그대로 따라 말하는 게 기본이에요. 질문은 짧고 친숙한 어휘로만 구성하세요.
🔁 이렇게 바꿔서 응용하세요

주제와 질문 문형만 바꾸면 어떤 서베이든 만들 수 있어요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
주제를 바꾸세요. "여행 경험", "가족 정보", "일과" 등.
조건
질문 문형과 난이도를 조정하세요. 학습자가 직접 질문을 만들면 난이도가 올라가요.
출력
인쇄할 때는 "A4 워크시트 형식"으로 정리해 달라고 하세요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 질문과 활동지 구조를 만들어 줘요. 파란색 키에 마우스를 올려보세요.

output.json
{ "questions": [ { "id": 1, "question": "이름이 뭐예요?", "pattern": "~이/가 뭐예요?" }, { "id": 2, "...": "..." } ], "table_template": { "columns": ["이름", "좋아하는 음식", "주말에 뭐 해요?"], "rows": 3 }, "summary_prompt": "가장 인기 있는 음식은 _____이에요." }
questions
→5개의 질문 목록이에요. 각 질문에는 ID와 사용 문형이 명시돼요.
table_template
→인터뷰 기록표의 열(columns)과 행(rows)을 정의해요.
summary_prompt
→정리 발표에 쓸 틀을 제시해요. 학습자가 빈 칸을 채우면 돼요.
⚠️ 질문이 정말 초급(TOPIK 1~2급) 수준인지 꼭 확인하세요. 낯선 어휘나 복잡한 문법이 들어가 있으면 활동이 원활하지 않아요.
🖨 Step 3. 인쇄용 활동지로 만드세요

JSON이 준비됐으면, 아래처럼 프롬프트를 추가해서 인쇄 가능한 형식으로 변환하면 돼요.

"이 데이터를 A4 인터뷰 활동지로 정리해줘. 질문은 굵게, 기록표는 큰 칸으로 해서 학습자가 쓰기 쉽게 해줘. 아래에 정리 발표 칸도 추가해줘."
