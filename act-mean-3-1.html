상황 카드는 학습자에게 "언제, 누구에게, 왜 이 말을 하는가"를 명시적으로 제시해요. 빈칸 채우기에서는 문장 내 문맥이 유일한 단서이지만, 상황 카드에서는 사회적 맥락(화자 관계, 장소, 목적)이 추가돼서 형태 선택의 근거가 풍부해지죠.

같은 의미라도 상황에 따라 적절한 표현이 달라지잖아요. "배가 아프다"를 친구에게 말할 때와 의사에게 말할 때 어떻게 다른지를, 두 장의 상황 카드로 나란히 보여주면 차이를 즉각 인식해요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
상황 설명이 구체적일수록 학습자가 표현을 선택하는 근거가 명확해져요. "친구"보다는 "20대 대학 친구", "병원"보다는 "처음 방문하는 큰 병원의 진료실"처럼 세부 정보가 있으면 좋아요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 3급 상황별 표현 선택 카드 생성 # 조건 (Constraints) - 카드 10장 - 각 카드: 상황 설명 + 표현 3개 (1적절, 2부적절) - 부적절 이유: 높임법, 격식, 직접성 등 화용 요소 - 상황: 학교, 직장, 병원, 식당, 친구 환경 섞기 - 각 선택지에 적절/부적절 여부 + 이유 포함 # 출력 (Output) JSON 형식
역할과 출력 형식은 그대로 두고, 과제와 조건만 바꾸면 어떤 문법 항목이든 만들 수 있어요.
🔁 이렇게 바꿔서 응용하세요

높임법 대신 다른 화용 요소로 만들고 싶으면, 과제와 조건만 수정하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 화용 항목에도 통해요.
과제
높임법 → 다른 화용 요소로 교체. 직접/간접, 완곡, 거절 등 다양한 언어 기능 다루기
조건
대상 레벨 유지. 3개 선택지 구조는 난이도 조절용 (줄이면 쉬워짐, 늘리면 어려워짐)
출력
앱이면 JSON, 인쇄용이면 "A4 카드 형식으로 정리해줘"로 변경
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 카드를 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "cards": [ { "id": 1, "situation": "교수님에게 과제 기한 연장 요청", "context": { "relationship": "비공식적 수직 관계 (교수-학생)", "formality": "높임법 필수", "place": "교수실" }, "options": [ { "text": "과제 내일까지 해 주세요.", "correct": false, "reason": "명령형으로 교수님에게 부적절" }, { "text": "과제 기한을 늘려 주실 수 있을까요?", "correct": true, "reason": "높임과 간접 표현으로 적절" }, { "text": "과제 내일 낼게.", "correct": false, "reason": "반말로 무례함" } ] }, { "id": 2, "...": "..." } ] }
situation
→상황 설명이에요. 무엇을 하려고 하는가를 명시해요.
context
→화자 관계·격식·장소를 담은 맥락이에요. 표현 선택의 근거가 되죠.
options
→선택지 3개: 적절한 표현 1개, 부적절한 표현 2개
correct
→선택지가 적절하면 true, 부적절하면 false예요.
reason
→적절/부적절 이유를 쓰는 곳이에요. 학습자 피드백으로 쓰여요.
⚠️ AI가 만든 카드는 꼭 확인하세요. 상황이 모호하거나 부적절한 표현의 이유가 불명확할 수 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
🛠 Step 3. 코드 생성 도구로 앱을 만드세요

"이 카드 데이터로 한국어 학습용 상황 이해 퀴즈 앱을 만들어줘. 상황 설명을 보여주고 3개 선택지 중 가장 적절한 표현을 고르면, 즉시 맞고 틀린지 표시하고 이유를 설명해줘."
