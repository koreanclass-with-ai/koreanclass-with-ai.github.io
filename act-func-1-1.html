규모역할 카드 4세트 / 15~20분
배포인쇄용 카드 세트

거절은 상대방의 요청이 있어야 성립하는 행위잖아요. 혼자서는 연습할 수 없고 상대역이 필요해요. 역할 카드는 A(요청하는 사람)와 B(거절하는 사람) 각각에게 상황, 관계, 목표를 제시해서 두 학습자가 자연스럽게 대화를 시작할 수 있게 해줘요.

거절의 핵심은 "같은 '아니요'도 상황에 따라 다르게 말해야 한다"는 거예요. 친구의 부탁을 거절할 때와 교수님의 제안을 거절할 때 사용하는 전략이 다르거든요. 역할 카드는 이 "상황에 따른 차이"를 물리적으로 보여줘요. 카드마다 화자 관계(친구/선후배/상사)와 부담도(작은 부탁/큰 부탁)가 다르게 설정되어 있어서, 학습자는 매 카드에서 "이 상황에서는 어떻게 거절해야 하지?"를 판단해야 해요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
B카드에 거절 이유와 대안을 "힌트"로만 주고 완성된 문장을 주지 않는 게 핵심이에요. 완성된 문장을 주면 학습자가 그대로 읽기만 하고, 스스로 거절 전략을 구성하는 연습이 되지 않거든요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 (화용론 전공) # 과제 (Task) TOPIK 3~4급 학습자를 위한 거절하기 역할 카드 세트 생성 # 조건 (Constraints) - 역할 카드 4세트 (각 세트: A카드 요청자 + B카드 거절자) - 난이도 순서: 친구+작은 부탁 → 친구+큰 부탁 → 선후배+작은 부탁 → 상사+큰 부탁 - 각 카드에: 화자 역할, 상황 설명, 대화 목표 - B카드에 거절 이유 힌트 제공 (키워드만, 완성된 문장 아님) - 각 세트에 거절 전략 해설 포함 # 출력 (Output) JSON 형식
4세트를 반드시 쉬운 것부터 배치해요. 친구에게 거절하는 것도 어려운 학습자에게 갑자기 상사에게 거절하는 상황을 주면 좌절감이 크거든요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 카드를 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "sets": [ { "id": 1, "difficulty": "쉬움", "relationship": "친구", "burden": "작은 부탁", "card_A": { "role": "친구", "situation": "토요일에 영화를 보고 싶어서 친구에게 같이 가자고 합니다.", "goal": "친구를 설득하여 영화를 같이 보러 가세요." }, "card_B": { "role": "친구", "situation": "토요일에 시험 공부를 해야 합니다.", "goal": "친구의 제안을 거절하세요.", "hint": "이유: 시험 공부 / 대안: 다음 주에 가자고 제안" }, "strategy_notes": "친구 관계이므로 반말 사용. 직접적 거절도 가능하나, 이유를 먼저 말하고 대안을 제시하면 더 자연스럽다." }, { "id": 2, "...": "..." } ] }
sets
→역할 카드 세트 목록이에요. 4세트가 들어가요.
difficulty
→난이도예요. 쉬움→보통→어려움 순서로 배치해요.
card_A
→A카드(요청자)의 역할, 상황, 대화 목표가 들어가요.
card_B
→B카드(거절자)의 역할, 상황, 목표, 거절 힌트가 들어가요.
hint
→거절 이유와 대안을 키워드로만 제시해요. 완성문은 금지!
strategy_notes
→거절 전략 해설이에요. 화용적 포인트를 설명해요.
⚠️ AI가 만든 카드는 꼭 확인하세요. 상황이 모호하거나 거절 전략의 설명이 불명확할 수 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
🛠 Step 3. 인쇄용으로 카드를 만드세요

JSON 데이터가 준비됐으면, AI에게 이렇게 말하면 돼요.

"이 JSON 데이터를 A4 인쇄용 역할 카드로 정리해줘. 각 세트마다 A카드와 B카드가 나란히 보이도록, 점선으로 잘라지는 부분을 표시해."

완성되면 PDF로 다운로드해서 바로 인쇄해 쓸 수 있어요!

PRACTICE
🎯 완성 자료 미리보기

역할 카드의 실제 모습을 확인해 보세요.

이 활동은 웹앱 데모 대신 완성된 자료의 모습을 보여줍니다.
A카드 (요청자)
역할:
친구
상황
토요일에 영화를 보고 싶어서 친구에게 같이 가자고 합니다.
목표
친구를 설득하여 영화를 같이 보러 가세요.
B카드 (거절자)
역할:
친구
상황
토요일에 시험 공부를 해야 합니다.
목표
친구의 제안을 거절하세요.
힌트 → 이유: 시험 공부 / 대안: 다음 주에 가자고 제안
거절 전략 해설: 친구 관계이므로 반말 사용. 직접적 거절도 가능하나, 이유를 먼저 말하고 대안을 제시하면 더 자연스럽다.
학생들은 A카드와 B카드를 각각 받고, 짝을 이루어 자유로운 대화를 진행합니다. 카드에 적힌 내용을 그대로 외우거나 읽는 게 아니라 자기 말로 상황을 이해하고 표현해야 해요.
CHECK
✅ 이것만 확인하세요
  • 4세트의 난이도가 순차적으로 올라가나요?
  • A카드와 B카드의 상황 설명이 서로 모순되지 않나요?
  • B카드의 힌트가 완성된 문장이 아닌 키워드 수준인가요?
  • 거절 전략 해설이 화용적으로 정확한가요? (특히 높임법)

바꾸고 싶은 것방법예시
기능 교체거절 → 다른 대인 관계 기능사과하기, 요청하기, 제안하기
난이도 올리기역할 카드 → 즉흥 시나리오교사가 구두로 상황만 설명, 카드 없이 즉석 대화

활동 확장거절로 끝남 → 합의 도출까지거절 후 절충안을 찾는 대화로 확장
