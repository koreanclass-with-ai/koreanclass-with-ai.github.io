눈으로 보고 고르는 것과 귀로 듣고 쓰는 건 완전히 다른 능력이거든요. 같은 문법이라도 채널을 바꾸면 다른 인지 경로를 거치게 돼요. 앞에서 선택형으로 시각적으로 연습한 항목을, 이번에는 청각적으로 다시 만나게 하면 학습의 깊이가 확 달라져요.

왜 빈칸 채우기가 음운·철자 연결에 적합하냐면요. 한국어는 발음과 철자가 다른 경우가 정말 많잖아요. 학교[학꾜], 같이[가치] 이런 식으로요. 학습자가 음성을 듣고 빈칸에 글자를 써넣으면, "들리는 소리 → 맞춤법에 맞는 철자"로 변환하는 연습이 자연스럽게 되는 거예요.

문장 전체를 받아쓰는 것보다 빈칸 하나에 집중하는 편이 초급 학습자한테 부담이 훨씬 적어요. 빈칸이라는 물리적 공간이 있으니까 "여기만 집중하면 돼"라는 신호가 되는 거죠.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) TOPIK 1~2급 학습자를 위한 듣기 빈칸 채우기 문항 생성 # 조건 (Constraints) - 문항 수: 8개 - 각 문장에서 핵심 어절 1개를 빈칸으로 처리 - 빈칸 위치는 문장의 끝이 아닌 중간에 배치 - 문장은 일상 대화 상황 (인사, 쇼핑, 식당 주문, 길 묻기 등) - 음운 변동이 포함된 어절을 2개 이상 포함 (예: 학교[학꾜], 같이[가치]) # 출력 (Output) JSON 형식
빈칸 위치를 문장 끝이 아닌 중간에 두는 이유가 있어요. 문장 끝은 문맥으로 예측이 가능하거든요. 중간에 두면 "들어야만 알 수 있는" 상황이 만들어져요.
audio_text 필드를 음성 생성 도구에 입력하면 TTS 음성이 생성돼요. 속도를 "느리게"로 설정하면 초급용, "보통"이면 중급용으로 쓸 수 있어요.
🔁 이렇게 바꿔서 응용하세요

듣기 빈칸 채우기 대신 다른 듣기 활동으로 만들고 싶으면, 과제와 조건만 수정하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 활동에도 통해요.
과제
듣기 활동의 목표에 맞게 수정하세요. "듣고 조사 채우기 문항 생성", "듣고 받아쓰기 문항 생성" 등으로요.
조건
레벨과 특성에 맞게 조정하세요. 중급이면 빈칸을 2개로 늘리고, 음운 변동 비율도 올릴 수 있어요.
출력
앱으로 만들 거면 JSON, 인쇄용이면 "교사용 대본 + 학생용 빈칸 워크시트 표 형식으로 정리해줘"로 바꾸세요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 문항을 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "questions": [ { "id": 1, "full_sentence": "저는 학교에서 한국어를 공부해요.", "display_sentence": "저는 ______에서 한국어를 공부해요.", "answer": "학교", "pronunciation_note": "학교[학꾜] — 경음화", "audio_text": "저는 학교에서 한국어를 공부해요." }, { "id": 2, "full_sentence": "우리 같이 식당에 갈까요?", "display_sentence": "우리 ______ 식당에 갈까요?", "answer": "같이", "pronunciation_note": "같이[가치] — 구개음화", "audio_text": "우리 같이 식당에 갈까요?" }, { "id": 3, "...": "..." } ] }
full_sentence
→빈칸 없이 완전한 문장이에요. TTS 음성 생성과 정답 확인에 사용돼요.
display_sentence
→빈칸이 포함된 문장이에요. 화면에 표시되는 형태예요.
answer
→빈칸에 들어갈 정답 단어예요. 학습자가 입력한 답과 비교해요.
pronunciation_note
→발음 노트예요. 음운 변동이 있으면 실제 발음과 규칙을 알려줘요.
audio_text
→TTS 음성 생성용 텍스트예요. 음성 생성 도구에 이 텍스트를 넣으면 돼요.
⚠️ AI가 만든 문항은 꼭 확인하세요. 음운 변동 표기가 잘못되었거나, 빈칸 위치가 문장 끝에 몰려 있을 수 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.
🛠 Step 3. 코드 생성 도구로 앱을 만드세요

JSON 데이터가 준비됐으면, Antigravity 같은 코드 생성 도구에 붙여넣고 이렇게 말하면 돼요.

"이 데이터로 한국어 듣기 빈칸 채우기 웹앱 만들어줘. 🔊 버튼 누르면 TTS 음성 재생되고, 학습자가 빈칸에 답 입력하면 정답/오답 표시되고, 발음 노트도 보여줘."

완성되면 아래처럼 작동하는 앱이 나와요. 직접 해보세요!
