'에'와 '에서'는 초급에서 가장 많이 헷갈리는 조사 중 하나예요. 설명은 이해했는데 막상 문장에서 고르려면 헷갈리는 거, 선생님들도 잘 아시죠?

선택형으로 만든 이유가 있어요. 초급 학습자는 한글 입력 자체가 부담이에요. 선택지를 두 개로 줄이면 "에/에서 중 뭐가 맞지?"라는 판단에만 집중할 수 있어요.

문항마다 해설이 다르게 나오는 것도 포인트예요. "'공부하다'는 행동 동사예요. 행동이 일어나는 장소에는 '에서'를 써요." 이런 식으로요. 이게 자동채점 웹앱의 가장 큰 장점이에요.

BUILD
🤖 Step 1. AI한테 재료를 만들어 달라고 하세요

Claude, ChatGPT, Gemini 아무거나 열고 아래 프롬프트를 그대로 붙여넣으세요.

📐 프롬프트를 잘 쓰는 법이 궁금하다면 → 프롬프트 작성 가이드 보기
⚠️ 이 활동에서 특히 주의하세요
'에'와 '에서'를 구분하는 문항을 만들어달라고 하면, AI가 이동의 '에'(학교에 가요)를 섞어서 내놓는 경우가 많아요. 아래 프롬프트처럼 "존재/행동 장소만, 이동의 에는 제외"라고 명시해줘야 원하는 결과가 나와요.
prompt.txt
# 역할 (Role) 한국어 교육 전문가 # 과제 (Task) 장소 조사 '에'와 '에서'를 구분하는 빈칸 채우기 문항 생성 // 존재/행동 장소만. 이동의 에(가다/오다)는 제외. # 조건 (Constraints) - 대상: TOPIK 1급 초급 학습자 - 문항 수: 10개 ('에' 5개, '에서' 5개 균형 배치) - 동사: 초급 기본 동사만 사용 (있다/없다/먹다/놀다/공부하다 등) - 장소: 학습자에게 친숙한 일상 장소 - 각 문항에 정답, 한국어 해설, 오답 해설 포함 # 출력 (Output) JSON 형식
역할과 출력 형식은 그대로 두고, 과제와 조건만 바꾸면 어떤 문법 항목이든 만들 수 있어요.
🔁 이렇게 바꿔서 응용하세요

에/에서 대신 다른 문법 항목으로 만들고 싶으면, 과제와 조건만 수정하면 돼요.

역할
그대로 두세요. 교육 전문가 역할은 어떤 문법 항목에도 통해요.
과제
목표 문법만 바꾸세요. "이/가 vs 은/는 구분 문항 생성", "과거 시제 어미 활용 문항 생성"
조건
대상 레벨과 문법 특성에 맞게 조정하세요. 중급이면 TOPIK 3급, 말하기 활동이면 대화문 형식으로.
출력
앱으로 만들 거면 JSON, 인쇄용이면 "표 형식으로 정리해줘"로 바꾸세요.
📤 Step 2. 결과를 확인하세요

AI가 아래처럼 JSON 형식으로 문항을 만들어 줘요. 직접 수정해서 쓸 수도 있어요. 파란색 키에 마우스를 올려보세요.

output.json
{ "questions": [ { "id": 1, "sentence": "저는 도서관______ 공부해요.", "answer": "에서", "verb_type": "행동", "feedback_correct": "✅ '공부하다'는 행동 동사예요.", "feedback_wrong": "❌ 행동 장소에는 '에서'를 써요." }, { "id": 2, "sentence": "학교______ 선생님이 있어요.", "answer": "에", "verb_type": "존재", "feedback_correct": "✅ '있다'는 존재 동사예요.", "feedback_wrong": "❌ 존재 장소에는 '에'를 써요." }, { "id": 3, "...": "..." } ] }
sentence
→빈칸이 포함된 문장이에요. ______가 빈칸 위치예요.
answer
→정답을 넣는 곳이에요. "에" 또는 "에서" 중 하나가 들어가요.
verb_type
→행동/존재/이동 중 해당하는 동사 유형이에요. 해설의 기준이 돼요.
feedback_correct
→학습자가 정답을 골랐을 때 나오는 해설이에요.
feedback_wrong
→학습자가 오답을 골랐을 때 나오는 해설이에요.
⚠️ AI가 만든 문항은 꼭 확인하세요. 어색한 표현이나 수준에 맞지 않는 어휘가 섞여 있을 수 있어요. AI는 빠르게 초안을 만들어 주는 조수고, 교육적 판단은 선생님 몫이에요.

"이 데이터로 한국어 학습용 빈칸 채우기 웹앱 만들어줘. '에' 또는 '에서' 선택하면 즉시 정답/오답 표시되고, 해설 나오고, 10문항 다 풀면 점수 보여줘."
