① 디자인
색·글꼴·버튼 모양을 정하는 부분. 겉모습을 바꾸려면 여기를 고쳐요.
<!-- 화면 스타일 --> <style> :root { --bg: #f5f7fb; /* 배경색 */ --primary: #2563eb; /* 버튼·강조색 (파랑) */ --good: #15803d; /* 정답 표시 (초록) */ --bad: #b91c1c; /* 오답 표시 (빨강) */ } body { font-family: system-ui, sans-serif; /* 글꼴 */ background: var(--bg); } .app { max-width: 560px; /* 화면 최대 너비 */ border-radius: 22px; /* 모서리 둥글기 */ box-shadow: 0 12px 35px rgba(15,23,42,0.12); } .sentence { font-size: clamp(25px, 7vw, 36px); /* 문장 크기 */ font-weight: 800; } .choice { /* 선택 버튼 */ background: var(--primary); min-height: 72px; font-size: 30px; } .feedback.correct { background: #dcfce7; } /* 정답 배경 */ .feedback.wrong { background: #fee2e2; } /* 오답 배경 */ </style>② 문항 데이터
문장·정답·해설이 들어 있어요. 문항 내용을 바꾸려면 여기를 고쳐요. (활동 페이지에서 본 그 부분)
// 문항 목록 — 이 안의 내용만 바꾸면 문항이 달라져요 const questions = [ { sentence: "책이 가방___ 있어요.", // ___가 빈칸 answer: "에", // 정답 correct: "정답이에요. '있어요'는 있는 장소라 '에'.", wrong: "아니에요. 있는 장소를 말하므로 '에'가 맞아요." }, { sentence: "민수는 식당___ 밥을 먹어요.", answer: "에서", correct: "정답이에요. '먹어요'는 행동이라 '에서'.", wrong: "아니에요. 먹는 행동의 장소라 '에서'가 맞아요." }, // ... 같은 형식으로 문항이 이어져요 (총 10개) // 문항을 추가하려면 { } 묶음을 복사해 붙이면 돼요 ];③ 작동 로직
버튼을 누르면 채점하고 점수를 세고 다음 문제로 넘기는 부분. 보통 손댈 일이 없어요.
// 답을 골랐을 때 실행되는 부분 function chooseAnswer(selected) { const q = questions[current]; const isCorrect = selected === q.answer; // 고른 답과 정답 비교 if (isCorrect) score++; // 맞으면 점수 +1 // 빈칸에 정답 채우고, 해설 보여주기 feedback.innerHTML = isCorrect ? q.correct // 맞았을 때 해설 : q.wrong; // 틀렸을 때 해설 } // 마지막 문제까지 풀면 점수 화면 보여주기 function showResult() { scoreBox.textContent = score + " / " + questions.length; // 점수에 따라 다른 격려 메시지 } // '다음 문제' 버튼을 누르면 다음 문항으로 nextBtn.addEventListener("click", () => { current++; (current >= questions.length) ? showResult() : renderQuestion(); });
